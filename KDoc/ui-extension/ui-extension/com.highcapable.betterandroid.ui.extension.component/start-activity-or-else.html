Start an Activity using ComponentName.

This function auto obtains the Activity class name that the target app needs to start through PackageManager.getLaunchIntentForPackage.

  • The Manifest.permission.QUERY_ALL_PACKAGES permissions or a queries list declaration required when target sdk higher than 29.
