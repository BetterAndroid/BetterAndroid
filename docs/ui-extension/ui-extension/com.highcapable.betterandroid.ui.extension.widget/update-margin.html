updateMargin

fun View.updateMargin(@Px left: Int = marginLeft, @Px top: Int = marginTop, @Px right: Int = marginRight, @Px bottom: Int = marginBottom)

Updates this view's margin.

Same like View.updatePadding.

This view layout params need to be a ViewGroup.MarginLayoutParams.

Parameters

left

the left margin (px), default is marginLeft.

top

the top margin (px), default is marginTop.

right

the right margin (px), default is marginRight.

bottom

the bottom margin (px), default is marginBottom.
