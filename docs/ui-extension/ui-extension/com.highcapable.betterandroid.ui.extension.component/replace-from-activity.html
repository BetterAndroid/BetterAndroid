replaceFromActivity

@JvmOverloads
fun Fragment.replaceFromActivity(activity: FragmentActivity, view: View? = activity.rootView(), @IdRes viewId: Int = -1, @AnimRes enterAnimId: Int = R.anim.simple_ui_open_in, @AnimRes exitAnimId: Int = R.anim.simple_ui_open_out, tag: String = System.currentTimeMillis().toString(), runOnCommit: Runnable? = null, isAllowingStateLoss: Boolean = true)

Use the current Fragment from FragmentActivity to replace the existing one.

Parameters

activity

the FragmentActivity that needs to be replace to.

view

the container that needs to be replace to, default is Activity.rootView.

viewId

the container view id that needs to be replace to.

enterAnimId

the Fragment start to enter animation, default is R.anim.simple_ui_open_in.

exitAnimId

the Fragment finished enter animation, default is R.anim.simple_ui_open_out.

tag

the Fragment tag, default is System.currentTimeMillis.

runOnCommit

the commit Runnable, default is null.

isAllowingStateLoss

whether to allow state loss, default true.

Throws

IllegalStateException

if view is null.
